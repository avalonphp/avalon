# Avalon

This repository contains the core components of the Avalon framework that powers
the open source project management and issue tracking system [Traq][1].

## In this repository

- HTTP
- Routing
- Kernel

## Other components

- [Database][2]
- [Helpers][3]
- [Hook][4]
- [Language][5]
- [Templating][6]
- [Validation][7]
